## 📌 Project Overview
This project focuses on Exploratory Data Analysis (EDA) of crime statistics data to identify patterns, trends, and insights related to criminal activities. The goal is to understand how crime varies across different regions, time periods, and categories.

The analysis helps in identifying high-risk areas, temporal trends, and dominant crime types using data visualization and statistical techniques.

---

## 📂 Dataset Information

The dataset includes features such as:

- Unit
- Name
- Year
- Dacoity
- Robbery
- Murder
- Speedy
- Trial
- Riot
- Woman & Child Repression
- Kidnapping
- Police Assault
- Burglary
- Theft
- Other Cases
- Arms Act
- Explosive	Narcotics
- Smuggling
- Total Cases  

---

## 🧹 Data Preprocessing

The following steps were performed:

- Handled missing values  
- Removed duplicate records  
- Converted date fields into proper format  
- Standardized categorical variables  
- Feature engineering for time-based analysis
